# Converged Learning Roadmap

This roadmap builds both skill sets simultaneously, with each phase reinforcing the connection between AI and security. Projects are sequenced so that knowledge compounds.

## Phase 1: Dual Foundations (Weeks 1–4)

**Goal:** Establish working knowledge in both fields and see the first convergence points.

- **Study:** Anomaly Detection Theory
- **Build:** Project 4 (Phishing Detection) — accessible entry point that uses core NLP + security concepts
- **Build:** Project 1 (Network Intrusion Detection) — the canonical AI-for-security project
- **Study:** Security-Aware ML System Design — start thinking about security from day one

## Phase 2: Deepening Both Sides (Weeks 5–8)

**Goal:** Build intermediate skills and tackle more complex convergence projects.

- **Study:** Adversarial ML Theory + LLM Security Fundamentals
- **Build:** Project 2 (Malware Classification) — deeper ML + real-world security domain
- **Build:** Project 7 (Adversarial ML Lab) — understand the attack surface of AI itself
- **Tackle:** Scenario 2 (Securing Medical AI) — apply your growing knowledge to a realistic assessment

## Phase 3: Advanced Convergence (Weeks 9–12)

**Goal:** Build advanced systems and start specializing.

- **Build:** Project 3 (Threat Intelligence NLP) — advanced NLP meets security operations
- **Build:** Project 8 (LLM Security) — the most in-demand niche right now
- **Build:** Project 5 (UEBA) — advanced behavioral analytics
- **Study:** Privacy & AI Ethics in Security Contexts
- **Tackle:** Scenario 1 (APT Hunt) or Scenario 4 (Compromised ML Pipeline)

## Phase 4: Mastery & Portfolio (Weeks 13–16)

**Goal:** Complete capstone projects that demonstrate end-to-end expertise.

- **Build:** Project 9 (Model Supply Chain Security) — emerging critical need
- **Build:** Project 10 (Privacy-Preserving ML) — regulated industry enablement
- **Build:** Project 11 (SOC Analyst Assistant) — the capstone integration project
- **Tackle:** Scenario 3 (AI Red Team Program) — demonstrate strategic thinking

## Phase 5: Differentiation (Weeks 17+)

**Goal:** Establish unique professional identity at the convergence.

- **Build:** Project 12 (Adversarial ML CTF) — community contribution that builds reputation
- **Tackle:** Scenario 5 (Deepfake Defense) — cutting-edge real-world challenge
- **Build:** Project 6 (Vulnerability Prioritization) — rounds out the defensive portfolio
- **Contribute** to open source: submit detection rules, adversarial ML tools, or security evaluations
- **Publish:** Write up findings as blog posts or conference talks (BSides, DEF CON AI Village, MLSecOps Community)
