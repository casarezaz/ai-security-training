# Project 3: Threat Intelligence NLP Pipeline

> **Track A:** AI-Powered Security Systems

| | |
|---|---|
| **Convergence** | AI/ML 65% · Cyber 35% |
| **Difficulty** | Intermediate |
| **Duration** | 10-14 hours |

## Overview

Build an NLP system that automatically ingests, processes, and structures threat intelligence from unstructured sources. Extract IOCs, TTPs, and actor attributions using NER and relation extraction.

## Why This Convergence Matters

Security teams drowning in unstructured threat data. NLP transforms raw text into structured, actionable intelligence that can be automatically processed and correlated.

## Learning Objectives

- Security-domain NLP and entity recognition
- Fine-tune language models on cybersecurity text
- Implement NER for IOCs (IPs, hashes, domains)
- Extract MITRE ATT&CK techniques from text
- Build knowledge graphs for threat actor attribution

## Implementation Steps

1. Curate corpus of threat intelligence reports from public sources
2. Implement regex + ML hybrid IOC extraction
3. Fine-tune BERT-based NER model on cybersecurity entities
4. Build relation extraction model for threat actor associations
5. Implement automatic MITRE ATT&CK technique tagging
6. Build knowledge graph (Neo4j) for threat correlations
7. Create API that returns STIX 2.1 structured bundles
8. Build dashboard showing threat landscape trends

## Deliverables

- Automated IOC extraction pipeline
- Fine-tuned NER model for cybersecurity entities
- Threat intelligence knowledge graph
- STIX 2.1 structured output API

CTI Analyst and Threat Intelligence Engineer roles, senior positions $160-200K+.
