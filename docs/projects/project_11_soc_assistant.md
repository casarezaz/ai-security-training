## Overview

Build an AI assistant that augments SOC analysts by automatically triaging alerts, enriching indicators, correlating events, suggesting investigation paths, and drafting incident reports. Capstone project tying together NLP, anomaly detection, threat intelligence, and security operations.

## Why This Convergence Matters

SOC analysts face alert fatigue and high burnout. AI assistant handles tier-1 triage, enrichment, and correlation, freeing analysts to focus on complex investigations and response.

## Learning Objectives

- Integrate NLP for alert summarization and classification
- ML for intelligent alert triage and severity scoring
- Knowledge graphs for threat correlation
- RAG (Retrieval-Augmented Generation) for investigation guidance
- Feedback loops for continuous improvement

## Implementation Steps

1. Build alert ingestion pipeline normalizing alerts from multiple sources (SIEM, EDR, IDS)
2. Implement ML triage model for alert prioritization
3. Build automated enrichment service (VirusTotal, AbuseIPDB, WHOIS, threat feeds)
4. Implement alert correlation engine for event threading
5. Build RAG-powered investigation assistant (search playbooks and knowledge base)
6. Implement natural language report generation
7. Build analyst feedback loop for model refinement
8. Create performance dashboard (MTTD, MTTR, analyst satisfaction)

## Deliverables

- Complete SOC assistant with multi-source alert ingestion
- ML-powered alert triage and prioritization
- Automated indicator enrichment pipeline
- Alert correlation engine for event threading
- RAG-powered investigation assistant
- Analyst feedback integration and performance dashboard

## Career Impact

Portfolio-defining project demonstrating end-to-end AI/Security integration. $160-220K+ skill set.
