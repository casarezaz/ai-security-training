## Overview

Systematically attack and defend image classifiers using adversarial examples. Implement white-box attacks (FGSM, PGD, C&W), black-box attacks, and defenses.

## Why This Convergence Matters

Understanding adversarial ML is essential for both building robust AI systems and understanding their attack surface. As organizations deploy ML in security-critical roles, adversarial robustness becomes a security requirement.

## Learning Objectives

- Taxonomy of adversarial attacks and threat models
- Implement gradient-based and gradient-free attacks
- Measure robustness and defense effectiveness
- Deploy defenses and evaluate tradeoffs

## Implementation Steps

1. Train standard CNN on CIFAR-10 as target model
2. Implement FGSM (Fast Gradient Sign Method)
3. Implement PGD (Projected Gradient Descent)
4. Implement C&W (Carlini & Wagner) attack
5. Build black-box transfer attacks
6. Implement query-based black-box attacks
7. Deploy defenses: adversarial training, input preprocessing, randomized smoothing
8. Create robustness leaderboard and benchmark

## Deliverables

- Attack implementation library (FGSM, PGD, C&W)
- Defense implementations with robustness certification
- Robustness evaluation framework
- Visual gallery of adversarial examples and defenses
